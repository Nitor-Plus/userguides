2. Basic Usage

Operation Format Accepted Values Example Example returns
Fields {} Field Names only {sku}
Values [] ALL [aaddcc-1]
Contains :: ALL {sku}::[A] Skus containing “A”
Does not contain !: ALL {sku}!:[A] Skus not containing “A”
Wildcard - TBD * ALL {sku}::[A*] SKUS starting with “A”
Equal to == ALL {sku}==[aaddcc-1] Returns skus equal to “aaddcc-1”
In =* ALL {sku}=*[aaa,bbb,ccc] Returns any skus that contain the values in the comma separated input.
Not Equal != ALL {sku}!=[aaddcc-1] All skus but “aaddcc-1”
Greater than >> {title}>>[2] Returns all titles greater than 2
Less than << {title}<<[2] Returns all titles less than 2
Greater than or equal to >= {title}>=[2] Returns all titles greater than or equal to 2
Less than or equal to <= {title}<=[2] Returns all titles less than or equal to 2
Column to sort on sortField Field Names only SortField=title Data sorted by title column
Direction to sort sortDirection ASC, DESC sortDirection=ASC Sorts in ascending format

3. Example Filtering GET URL


Filter by one field

https://api.stok.ly/v0/items?filter={barcode}::[5000111022132]

Filter by two fields

https://api.stok.ly/v0/items?filter={barcode}::[5000111022132]&&{status}==[0]

ASCII HTML

https://api.stok.ly/v0/items?filter={barcode}::[5000111022132]%26%26{status}==[0]

Set Sort Field and Order
Ascending 

https://api.stok.ly/v0/items?sortField=name&sortDirection=ASC

Descending 

https://api.stok.ly/v0/items?sortField=name&sortDirection=DESC

Set Page Size & Offset Number

https://api.stok.ly/v0/items?size=100&page=0

Joingin Multiple Filters

https://api.stok.ly/v0/items?filter={name}::[adam]%26%26{status}==[0]&sortField=name&sortDirection=ASC&size=100&page=0

2. Items - Get an item

GET

https://api.stok.ly/v1/items/:id

Example: 

https://api.stok.ly/v0/items/f16c06bc-f076-4bfc-9999-cc3de6a08f0c/

Field Type Description
id String Items unique identifier
JavaScript Fetch Example



var myHeaders = new Headers();
myHeaders.append("Authorization", "Bearer *EXAMPLE TOKEN STRING HERE*");
                                                        
var requestOptions = {
    method: 'GET',
    headers: myHeaders,
    redirect: 'follow'
};
                                                        
fetch("https://api.stok.ly/v0/items/f16c06bc-f076-4bfc-9999-c13de6a08f0c/", requestOptions)
    .then(response => response.text())
    .then(result => console.log(result))
    .catch(error => console.log('error', error));
